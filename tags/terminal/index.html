⌘ Terminal Navigation

16-04-2019 — by Chanunnaki — 4 min read
#terminal  #tutorial  #macOS 
The Terminal, (AKA CLI, or “Command-Line Interface”) is super-powerful and allows you to interface directly with the underlying workings of your operating system. It’s pretty essential you learn how to use the Terminal if you’re going to be developing software as many operations are done exclusively through the Terminal. In this post, I’ll give you a quick run-down on how to navigate your file system in Terminal. At the end, there is a bonus tip for altering the appearance of your Terminal to make things more efficient for you.
